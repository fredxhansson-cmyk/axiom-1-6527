# Automatisk återvinningsschemaläggare

## Kort deploy-guide

1. Klona repositoriet.
2. Skapa en `.env.local` fil baserat på `.env.example`.
3. Fyll i dina Stripe API-nycklar.
4. Kör `npm install` för att installera beroenden.
5. Använd `npm run dev` för att starta utvecklingsservern.
6. Bygg projektet med `npm run build`, kör det sedan med `npm run start` i produktionsläge.
